command line tools

  • ep is the second generation of an eprint harvester tool, it will write harvest content to a dataset
  • eputil is the new harvester, it only talks to the REST API with the option of returning a JSON document, this can be piped into a dataset if needed
  • doi2eprintxml is a CaltechAUTHORS centric DOI to EPrint XML document generator

Roadmap for eputil

  • eputil add support for importing content to an EPrint repository
  • eputil add support for updating existing EPrint records (e.g. adding ORCID to creator items for a given author)
