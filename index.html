eprinttools

This project contains the eprinttools, a go package for working with EPrints REST API. It also includes ep and command line utility for harvesting content into a dataset collection and rendering JSON documents for web feeds.

The command line

  • eputil is a command line utility for interacting (e.g. harvesting) JSON and XML from EPrints’ REST API
    • uses minimal configuration because it does less!
    • will supercede ep
  • ep is an older version of a EPrints harvester. It features an integration with dataset and can be used to produce alternative feeds and formats.

The ep utility is configured from the environment or command line options. The environment settings are overridden by command line options. For details running either command envoke the tool name with the ‘-help’ option.
