## environment vars

+ EPGO_URL - base URL for your eprints repository (e.g. http://lemurprints.local)
+ EPGO_USERNAME - username required to access API( e.g. janedoe)
+ EPGO_PASSWORD - password required to access API (e.g. janesecret)
